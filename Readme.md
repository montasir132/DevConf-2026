# 🌐 DevConf 2026

DevConf 2026 is a premier developer conference bringing together engineers, founders, and builders from around the world.  
This project is a **practice assignment website** built with **HTML5 + CSS3** (no JavaScript, no frameworks).  

---

## 🚀 Features

### 🔹 Navbar
- Logo aligned to the left ("DEVCONF 2026")
- Menu items centered (Speakers, Schedule, Tracks, Venue, Blog)
- "Register Now" button aligned to the right

### 🔹 Hero Banner
- Full-width dark background image with overlay
- Bold heading: `Code. *Connect*. Create`
- Supporting paragraph text
- Centered "Register Now" button

### 🔹 Meet the Speakers
- Centered heading: "Meet the Speakers"
- 2x2 grid of speaker cards
- Each card includes:
  - Speaker photo
  - Track label (AI/ML, Cloud & DevOps, Frontend, Security)
  - Speaker name
  - Title/company

### 🔹 Secure Your Spot (Pricing)
- Centered heading + subtitle
- Three pricing cards (Standard, Pro, Team)
- Middle plan (Pro) visually highlighted
- Each card includes plan name, price, features, and button

### 🔹 About Section
- Conference introduction
- Highlights (Days, Speakers, Workshops)

### 🔹 Why Join DevConf 2026?
- Networking, Learning, Innovation, Opportunities
- Hover effects with CSS

### 🔹 Footer
- Logo on the left
- Social media icons on the right
- Divider line
- Terms of Service + Privacy Policy links
- Copyright
